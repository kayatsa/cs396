## A7: Make-A-Star

Array of floats

### Features

**POINTS:** Number of points the star has

**SPIKINESS:** Length of each point

**TWINKLE:** Amount each point's length increases and decreases

**SPIN:** Rotational speed

**LUMINOSITY:** Opacity
